본 프로젝트는 로컬 웹 서버 환경에서 실행되도록 제작되었습니다.

브라우저에서 HTML 파일을 직접 실행할 경우 CORS(Cross-Origin Resource Sharing) 보안 정책으로 인해 glb, obj, hdr, txt 등의 리소스가 정상적으로 로드되지 않을 수 있습니다.

- Live Server Extension

3. exhibition.html 또는 scene_control_editor.html 파일을 우클릭합니다.
4. "Open with Live Server"를 선택하여 실행합니다.

Eng(영어)

How to Run

This project is designed to run on a local web server.

Opening the HTML files directly in a browser may prevent assets such as glb, obj, hdr, and txt files from loading correctly due to CORS (Cross-Origin Resource Sharing) restrictions.

Recommended Environment

- Visual Studio Code
- Live Server Extension

Steps

1. Open the project folder in Visual Studio Code.
2. Install the Live Server extension.
3. Right-click either exhibition.html or scene_control_editor.html.
4. Select "Open with Live Server" to run the project.
